# 更新日志

## v2.26.0 (2026-06-20)

### 回测引擎全面重构 — 引入专业量化校验体系

#### 致命缺陷修复 (P0)
- **前视偏差修复**: T日收盘信号 → T+1日开盘价执行（旧实现用同日收盘价成交，系统性高估收益）
- **涨跌停模拟**: 涨停买不进/跌停卖不出，自动跳过不可执行交易，统计跳过数量

#### 高优先级增强 (P1)
- **完整交易成本**: 佣金万2.5 + 印花税千1(卖) + 过户费万0.2 + 滑点0.1%，乘法模型精确扣除
- **数据前复权验证**: 检查涨跌幅一致性，未复权时自动警告
- **脏数据过滤**: 跳过停牌(零量连续)和一字板(开=高=低=收)信号，排除数据质量干扰

#### 中优先级完善 (P2)
- **过度拟合检测**: Walk-Forward滚动窗口 + 分窗口标准差 + IS/OOS对比 + 过拟合判定
- **仓位管理分析**: 检测连续亏损幅度递增的马丁加仓模式
- **K线校验工具集**: `KlineValidator` 提供涨停/跌停/一字板/停牌/前复权检测

#### 架构重构
- **通用回测执行器** `_runGenericBacktest`: 消除6个策略~200行重复代码
- **回测配置类** `BacktestConfig`: 可配置成本/涨跌停/过滤开关，支持A股主板/科创板/旧版兼容
- **校验元数据** `BacktestValidationMeta`: 每笔回测附带完整的校验状态
- **Walk-Forward结果** `WalkForwardResult`: IS/OOS收益、分窗口标准差、过拟合判定

#### 回测校验报告
- `validationReport()`: 生成10项专业校验报告（未来函数/马丁加仓/过度拟合/完整成本/复权除权/前视偏差/幸存者偏差/涨跌停模拟/交易日历/脏数据）
- 每项标注通过/警告/未启用状态，附详细统计

- **H1**: ATR止损日内 vs T+1 事件文档化

- **H3**: minCommission未建模警告
- **M4**: 成本模型减法→乘法
- **M3**: windowStdDev/windowReturns字段正确重命名
- **M6**: profitFactor 单次遍历计算优化

### 测试体系
- **回测验证**: 原有95个测试全通过，新增23个校验专项测试

## v2.25.0 (2026-06-17)

### 核心优化：短线交易策略重构

- **信号检测增强**: 新增8种多日K线形态（阳包阴/阴包阳、三阳开泰/三只乌鸦、启明星/黄昏星、十字星），提高短线信号覆盖面
- **止损升级为ATR动态**: 基于ATR×2动态止损 + 20日低点摆动止损 + MA60长线防线，替代固定百分比止损
- **分级止盈**: TP1/TP2/TP3三级目标，支持追踪止损参数
- **回测反馈闭环**: megaBacktest全6策略回测 + 策略置信度调整(0.7~1.3) + 双向信号反馈(卖出信号反向压低置信度)
- **探索引擎性能优化**: K线批量预取 + 行情批量请求，分析阶段0网络IO，请求数减少95%
- **交易仪表盘**: 用"决策"Tab替换"分析"Tab，一屏聚合行情/信号/评分/止损/止盈/回测摘要
- **置信度权重重平衡**: 基本面权重降至10%，市场环境+信号时效性升至20%，适配短线决策

### Bug修复

- **评分映射确定性**: `.round()` → `.floor()`，消除5.5边界歧义
- **买卖方向偏差**: 修复buyCount==sellCount时置信度偏向卖出的bug
- **Duration序列化修复**: `toJson()`写索引(0,1,2)但`_parseDuration`读(2,10,30)导致往返损坏
- **回测零除守卫**: 所有backtest方法通用`_safeReturnPct()`，profitFactor分母守卫
- **估值过滤逻辑修正**: PE≤0 (亏损股) 不再通过探索筛选
- **KDJ超卖检测**: J<0 → J<20，修复阈值过于极端不可达的bug
- **跌停异常保护**: risk_pct除以零守卫(entryLow>0)

### 测试体系

- **回测验证测试**: 25个综合回测测试覆盖6种策略4种市场环境
- **边界条件测试**: 空K线/单K线/全同价/递减K线数全链路验证
- **混沌测试**: 衰减曲线/极限组合fuzz/自适应阈值
- **伪反转测试**: 鞭梢/假突破/头肩假象/多重鞭梢对倒免疫
- **极限行情**: 跌停/涨停/涨跌停交替异常测试
- **对倒测试**: 集合竞价大单对倒+多周期协同衰减拟合
- **预存失败修复**: 50个预存测试全部修复，新增32个回测专项测试
- **完整测试**: 490 tests / 0 failures

## v2.22.0 (2026-06-14)

### 关键修复

- **回测引擎崩溃修复**: 全胜场景下 profitFactor 为 double.infinity 导致 toStringAsFixed() 崩溃，改用 999.99 表示极大值
- **HTTP连接池泄漏修复**: MarketContextProvider 每次请求创建新 http.Client 未关闭，改为静态复用单例
- **单例 dispose() 后不可用修复**: ExploreEngine/OpportunityEngine/SectorPickEngine 的 StreamController 关闭后无法重建，增加 _ensureController() 自动恢复机制

- **行情客户端重构**: WebSocketClient 重命名为 QuotePollingClient，改为批量轮询模式，延迟初始化 HTTP Client
- **信号引擎优化**: 提取辅助函数，增加多空对抗验证反馈
- **K线数据校验**: API 数据增加 DataValidator 校验
- **策略面板重构**: 提取 StrategyPanelConfigured 公共类，消除重复代码
- **搜索防抖**: 搜索页面增加 debounce Timer 并正确 dispose
- **交易日判断修正**: 增加补班日处理逻辑 _isMakeupDay()

### 细节修复
