# UI重构设计：发现页面融合 + 自选升级 + 整体风格优化

> 版本: v2.20.0 | 日期: 2026-06-14 | 方案: A（渐进融合）

## 1. 概述

将机会界面与探索界面融合为"发现"页面，自选界面全面升级，整体UI采用现代简约风格重新设计。

### 核心变更

1. **导航结构**：6个Tab → 5个Tab（首页/自选/发现/资讯/留档）
2. **发现页面**：融合机会+探索，顶部Tab切换（自选分析 | 智能探索）
3. **自选界面**：卡片式布局 + 左滑删除 + 实时行情 + 分组筛选
4. **UI风格**：现代简约（类雪球/富途），深色主题重新设计

## 2. 导航结构

### 底部导航（5个Tab）

| 索引 | 图标       | 标签 | 页面              | 变更             |
| -- | -------- | -- | --------------- | -------------- |
| 0  | home     | 首页 | HomeScreen      | 不变             |
| 1  | star     | 自选 | WatchlistScreen | 全面升级           |
| 2  | explore  | 发现 | DiscoverScreen  | **新**（融合机会+探索） |
| 3  | article  | 资讯 | NewsScreen      | 不变             |
| 4  | bookmark | 留档 | ArchiveScreen   | 不变             |

### 删除的页面

* `OpportunityScreen` → 功能合并到 DiscoverScreen 的"自选分析"Tab

* `ExploreScreen` → 功能合并到 DiscoverScreen 的"智能探索"Tab

## 3. 发现页面（DiscoverScreen）

### 3.1 整体结构

```
┌─────────────────────────────┐
│  AppBar: "发现"              │
├─────────────────────────────┤
│  [自选分析] [智能探索]  ← 胶囊Tab切换器
├─────────────────────────────┤
│  状态栏（进度/时间/数量）     │
├─────────────────────────────┤
│  筛选/排序条                 │
├─────────────────────────────┤
│  卡片列表                    │
│  ...                        │
├─────────────────────────────┤
│  底部操作区                  │
└─────────────────────────────┘
```

### 3.2 自选分析Tab

**功能**：分析自选股的买卖机会和风险，替代原OpportunityScreen。

**布局**：

* 状态栏：分析进度/上次分析时间 + 股票总数

* 操作栏：筛选条（全部/看多/看空/观望）+ 编辑按钮 + 一键留档按钮

* 列表：卡片式布局

  * 每张卡片：股票名称+代码、价格+涨跌幅（涨红跌绿）、推荐标签+评分、信号标签行

  * 卡片右上角：留档图标按钮

  * 非编辑模式：点击跳转QuoteScreen

  * 编辑模式：左侧出现勾选框，支持多选

* 编辑模式底部：已选N只 + "删除选中（移出自选）"按钮

* 底部：刷新分析按钮

**新增功能**：

* 编辑模式：多选后批量移出自选

* 卡片式布局替代分组列表

### 3.3 智能探索Tab

**功能**：智能选股发现新机会，替代原ExploreScreen。

**布局**：

* 状态栏：分析进度/上次分析时间 + 优质标的数量

* 操作栏：排序条（评分/涨幅/名称）+ 筛选条（全部/买入/观望）

* 列表：卡片式布局

  * 每张卡片：排名序号（前3名高亮）、股票名称+代码、推荐标签、价格+涨跌幅+PE+PB+评分

  * 卡片底部操作行：加自选按钮（未加→"加自选"，已加→"已加"灰色）+ 查看详情按钮

* 底部操作区：

  * 分析中：进度条 + "开始智能选股"按钮（禁用）

  * 分析完成：N只可加自选 + "一键加自选"按钮（批量添加所有未加自选的股票）

* 空状态：提示文案 + "开始探索"按钮

**新增功能**：

* 底部"一键加自选"按钮（批量添加所有未加自选的股票）

* 自选状态实时同步（从自选分析Tab删除后，切回探索Tab状态更新）

### 3.4 Tab间数据同步

* 自选分析Tab删除自选 → 智能探索Tab刷新自选状态集合

* 智能探索Tab添加自选 → 自选分析Tab标记需刷新

* 切换Tab时检查数据新鲜度，过期则自动刷新

## 4. 自选界面升级（WatchlistScreen）

### 4.1 布局

```
┌─────────────────────────────┐
│  搜索栏（圆角搜索框+添加按钮）│
├─────────────────────────────┤
│  筛选条：全部/看多/看空/观望  │
│  排序：涨跌幅/评分/名称/时间  │
├─────────────────────────────┤
│  ┌───────────────────────┐  │
│  │ 股票名称  代码         │  │
│  │ ¥价格   +涨跌额       │  │
│  │ 涨跌幅  PE PB  评分   │  │
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ ...更多卡片...         │  │
│  └───────────────────────┘  │
├─────────────────────────────┤
│  编辑模式底部栏（可选）      │
└─────────────────────────────┘
```

### 4.2 卡片内容

每张卡片显示：

* 左侧：股票名称（16sp粗体）+ 代码（12sp次色）

* 中间：实时价格（20sp粗体）+ 涨跌额 + 涨跌幅（14sp，涨红跌绿）

* 右侧：PE/PB（12sp次色）+ 评分标签（如"买入 8.5"）

### 4.3 交互优化

1. **左滑删除**：替代原来的删除按钮，左滑露出红色"删除"按钮
2. **长按编辑**：长按进入编辑模式，顶部出现全选/取消，底部出现操作栏
3. **编辑模式底部**：已选N只 + "删除选中"按钮
4. **点击跳转**：卡片点击跳转QuoteScreen
5. **实时刷新**：30秒自动刷新行情，下拉强制刷新
6. **涨跌色**：涨=红色(#E74C3C)，跌=绿色(#2ECC71)，平=灰色

### 4.4 筛选与排序

* 筛选：全部/看多/看空/观望（基于最新分析结果的推荐类型）

* 排序：涨跌幅/评分/名称/添加时间，支持升降序切换

## 5. 整体UI风格

### 5.1 色彩体系

| 用途   | 颜色   | 色值        |
| ---- | ---- | --------- |
| 背景   | 深蓝黑  | `#0D1117` |
| 卡片   | 微亮深蓝 | `#161B22` |
| 卡片边框 | 微妙边框 | `#30363D` |
| 强调色  | 蓝色   | `#58A6FF` |
| 涨    | 红色   | `#E74C3C` |
| 跌    | 绿色   | `#2ECC71` |
| 文字主色 | 亮白   | `#F0F6FC` |
| 文字次色 | 灰色   | `#8B949E` |

### 5.2 卡片设计

* 圆角：12px

* 边框：1px `#30363D`

* 内边距：16px

* 卡片间距：8px

* 点击动效：轻微缩放（scale 0.98）

### 5.3 字体规范

| 元素       | 大小   | 粗细 | 颜色   |
| -------- | ---- | -- | ---- |
| 股票名称     | 16sp | 粗体 | 主色   |
| 价格       | 20sp | 粗体 | 涨/跌色 |
| 涨跌幅      | 14sp | 普通 | 涨/跌色 |
| 代码/PE/PB | 12sp | 普通 | 次色   |
| 评分标签     | 12sp | 粗体 | 强调色  |

### 5.4 Tab切换器

* 胶囊式设计

* 选中态：填充强调色背景 + 白色文字

* 未选中态：透明背景 + 次色文字

* 切换动画：滑动过渡

### 5.5 底部导航

* 5个图标

* 选中态：强调色图标 + 文字 + 小圆点指示

* 未选中态：灰色图标 + 文字

### 6.3 DiscoverScreen内部架构

```dart
class DiscoverScreen extends StatefulWidget {
  // 内部Tab状态
  int _currentTab = 0; // 0=自选分析, 1=智能探索

  // 两个Tab共享的子组件
  // - 筛选条
  // - 卡片列表
  // - 底部操作区

  // 自选状态同步
  Set<String> _watchlistCodes = {};
  bool _analysisNeedsRefresh = false;
}
```

### 6.4 自选状态同步机制

* DiscoverScreen维护`_watchlistCodes`集合

* 自选分析Tab删除自选 → 更新集合 → 通知智能探索Tab

* 智能探索Tab添加自选 → 更新集合 → 标记自选分析需刷新

* 切换Tab时检查`_analysisNeedsRefresh`标志

### 6.5 左滑删除实现

使用`Dismissible`组件：

```dart
Dismissible(
  key: Key(stock.code),
  direction: DismissDirection.endToStart,
  background: Container(color: Colors.red, child: Icon(Icons.delete)),
  onDismissed: (_) => _removeFromWatchlist(stock.code),
  child: StockCard(stock: stock),
)
```

### 6.6 实时行情自动刷新

* Timer.periodic(30秒)调用ApiClient批量获取行情

* 页面可见时启动，不可见时暂停

* 价格变化时微动效（颜色闪烁）
