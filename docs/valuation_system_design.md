## 二、TechnicalScorer — 技术面评分 (权重 33%)

**文件**: `mobile/lib/analysis/technical_scorer.dart`

5维等权求和，总分 clamped [0, 10]。

### 2.1 信号评分 (0-3分)

根据买卖信号力量差计算：

```
buyStrength = Σ(buySignals.strength × ADX_trend_weight)
sellStrength = Σ(sellSignals.strength × ADX_trend_weight)
signalRaw = (buyStrength - sellStrength) / maxTotal × 3

其中:
  ADX > 25 (趋势市): MA/MACD信号 × 1.2
  ADX < 20 (震荡市): RSI/KDJ信号 × 1.2
  maxTotal = (buyStrength + sellStrength) × 0.6, clamped [30, 150]
```

**买入信号数量惩罚 (v2.60)**:

| 买入信号数 | 系数 | 数据依据 |
|-----------|------|---------|
| ≥5 | ×0.60 | 胜率33.3% |
| ≥4 | ×0.75 | — |
| ≥3 | ×0.88 | 胜率38.4% |

### 2.2 趋势评分 (0-2分)

| 条件 | 基础分 |
|------|--------|
| MA5>MA10>MA20 (多头排列) | 1.4 |
| MA5>MA10 (短期多头) | 1.1 |
| MA5>MA20 | 0.7 |
| 其他 | 0.3 |
| MA5<MA10<MA20 (空头) | 0 |

**惩罚/加成**:

| 条件 | 调整 |
|------|------|
| MA20偏离率 > 8% (追高) | ×0.85 |
| MA20偏离率 > 5% | ×0.92 |
| 日内暴跌 close < open × 0.95 | ×0.60 |
| 日内暴跌 close < open × 0.97 | ×0.80 |
| ADX>25 且价格>MA5 (趋势确认) | +0.5 |
| ADX<20 (盘整) | -0.3 |

### 2.3 动量评分 (0-2分)

根据ADX区分趋势市/震荡市，RSI不同解读：

**趋势市 (ADX>25)**:
| RSI范围 | 得分 |
|---------|------|
| ≥60 | 1.8 |
| 50-60 | 1.3 |
| 40-50 | 0.8 |
| <40 | 0.3 |

**震荡市 (ADX<20)**: 超卖反弹逻辑
| RSI范围 | 得分 |
|---------|------|
| <30 | 1.6 |
| 30-40 | 1.3 |
| 40-60 | 1.0 |
| 60-70 | 0.7 |
| ≥70 | 0.3 |

**普通市**: RSI6<40→1.4, 40-60→1.0, 60-70→0.8, ≥70→0.5

BIAS6乖离惩罚: |bias|>5 → -0.4, |bias|>3 → -0.2

### 2.4 量价评分 (0-1.5分)

基于量比(volume/volMa5) + OBV趋势：

| 方向 | 量比>1.5 | 量比>1.0 | 量比≤1.0 |
|------|---------|---------|----------|
| 收阳 | 1.4 | 1.1 | 0.6 |
| 收阴 | 0.2 | 0.5 | 0.8 |

OBV5日趋势确认: 价涨量涨 +0.3, 价跌量跌 -0.2

### 2.5 波动率评分 (0-1.5分)

ATR%倒U型：最佳2-5%

| ATR% | 得分 |
|------|------|
| <1% | 0.3 (波动不足) |

## 四、ConfluenceScorer — 共振评分 (权重 12%)

**文件**: `mobile/lib/analysis/confluence_scorer.dart`

10指标方向一致性评分，基准5.0，bull加分/bear减分各上限5。

### 4.1 指标定义

| 指标 | 多头条件 | 空头条件 | 权重 |
|------|---------|---------|------|
| **MA** | MA5>MA10>MA20 | MA5<MA10<MA20 | 1.5 |
| **MACD** | DIF>DEA 且柱>0 | DIF<DEA 且柱<0 | 1.5 |
| **RSI** | RSI6 > 60 | RSI6 < 40 | 0.8 |
| **KDJ** | K>D 且 K<80 | K<D 且 K>20 | 0.8 |
| **BOLL** | 收于中轨上方 | 收于中轨下方 | 1.0 |
| **VOL** | 放量收阳 | 放量收阴 | 1.2 |
| **WR** | >80 (超卖看多) | <20 (超买看空) | 0.6 |
| **CCI** | >100 | <-100 | 0.6 |
| **GAP** | 向上跳空 | 向下跳空 | 0.4 |
| **DIVER** | 底背离 | 顶背离 | 1.0 |

### 4.2 计算公式

```
weightedBull = Σ(多头指标权重).clamp(0, 5)
weightedBear = Σ(空头指标权重).clamp(0, 5)
score = (5.0 + weightedBull - weightedBear).clamp(0, 10)
```

## 九、SuggestionGenerator — 操作建议

**文件**: `mobile/lib/analysis/suggestion_generator.dart`

### 9.1 动态仓位公式

```
仓位% = 基础仓位 × 等级系数 × (0.5 + 置信度) × 熊市因子

等级系数:
  强烈买入: 0.45×基础
  买入(≥3信号且≥8分): 0.30×基础
  买入(其他): 0.15×基础
  谨慎买入: 0.15×基础

熊市: ×0.5
最小: 3%, 最大: 40%
```

### 9.2 止损参考

- 优先: MA20 (20日均线)
- 备选: 10日最低价

## 十、数据流与调用关系

```
generateAnalysis(klines, quote, marketContext, newsList, sector)
  │
  ├─ SignalLayer.detectAllSignals(klines)        → buySignals, sellSignals
  ├─ PatternRecognizer.detectAll(klines)          → 形态信号追加
  ├─ MarketStructureAnalyzer.analyze(klines)      → marketStructure (5种)
  ├─ PercentileAnalyzer.analyze(klines, quote)    → PE/PB percentile
  ├─ LimitUpAnalyzer.analyzeFromDaily()           → 打板分析
  │
  ├─ TechnicalScorer.score(klines, buy, sell)     → techResult (0-10)
  ├─ RealtimeScorer.score(quote)                   → realtimeScore (0-10)
  ├─ ConfluenceScorer.score(last, signals)         → confluenceResult
  ├─ CapitalFlowAnalyzer.analyze()                 → capitalFlowScore (0-10)
  │
  ├─ ComprehensiveScorer.combine(...all...)        → compResult
  │     └─ FundamentalAnalyzer.analyze(quote)      → PE/PB/ROE评分
  │     └─ NewsSentimentAnalyzer.analyze(news)     → 新闻情绪
  │
  ├─ RiskAnalyzer.analyze(klines, last, quote)     → riskResult
  ├─ OpportunityIdentifier.identify(buySignals)    → opportunities
  ├─ BacktestEngine.megaBacktest(klines)           → 6策略回测
  ├─ StrategyBuilder.buildLayeredStrategies()      → 分层策略
  ├─ ConfidenceCalculator.calculate(...)           → confResult
  ├─ _generateReasons(...)                         → reasons + 推荐摘要
  ├─ SuggestionGenerator.generate(...)            → suggestions
  └─ RecommendationTracker.track()                 → 追踪推荐 + dimension_scores_json
```

推荐统计页独立读取已关闭推荐：

```
recommendation_tracking(is_closed=1, day20_return IS NOT NULL)
  → WeightOptimizer.getDimensionPerformanceReport()
  → 维度表现 / 建议权重 UI
```
